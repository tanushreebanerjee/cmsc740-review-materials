# CMSC 740 Review Material

This repository contains review materials for the CMSC 740 final exam.

## Contents

- `pdfs/` - Course lecture PDFs covering topics from introduction through advanced rendering and 3D geometry processing

## Course Topics

Based on the lecture materials, this course covers:

1. Introduction
2. Acceleration structures
3. Radiometry
4. Monte Carlo integration
5. BRDF and reflection integral
6. Rendering equation
7. Advanced sampling techniques (MIS, beyond uniform random sampling)
8. Three point form, BDPT
9. Participating Media and Subsurface Scattering
10. Denoising
11. Learning to Importance Sample in Primary Sample Space
12. Neural Importance Sampling
13. Neural radiosity
14. Introduction to 3D geometry processing
15. 3D point clouds - surface reconstruction
16-18. Scene reconstruction from images
19-21. Data-driven shape modeling (deep learning, generative modeling)
